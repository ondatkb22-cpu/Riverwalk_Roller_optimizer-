Framework Overview

This tool uses a multi-factor composite algorithm to build mathematically optimal lineups.

  1. Simulate: The engine scores players using a Composite Alpha: Win Equity, Value ($DLTA), Course Fit (SG), and Velocity.
  2. Lock: Browse the Arb tab for CORE grade players and tap + to lock your favorites.
  3. Pre: Instantly view 3 dynamically pre-built optimal lineups (Steam, GPP, Cash) perfectly tailored to the latest live odds.
  4. Optimize: Tap the floating AUTO button to execute a Search Procedure and return the highest-scoring lineup strictly between $48,500 and $50,000.
